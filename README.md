## 🚀 Rodar Shortlink

Para rodar o shortlink, siga estas etapas:

```
# clone o repositório

cd shortlink

# API
cd short_link_api

yarn

yarn dev

# Abra outra aba do projeto em seu terminal

# React project
cd short_link_web

yarn

yarn dev

```
